# TestWebRL - Setup & Build Instructions

## Prerequisites

- **Node.js** 20.x (20.11+ works, 20.19+ recommended) or 22.12+
- **npm** (comes with Node.js)

Download Node.js from https://nodejs.org/

## First-time setup

```bash
cd TestWebRL
npm install
```

## Development

Start the dev server with hot reload:

```bash
npm run dev
```

This opens a local server (usually http://localhost:5173). Changes to source files reload automatically.

## Production build

```bash
npm run build
```

Output goes to the `dist/` folder.

To preview the production build locally:

```bash
npm run preview
```

## Deploying to itch.io

1. Run `npm run build`
2. Zip the **contents** of `dist/` (not the folder itself — `index.html` must be at the root of the zip)
3. Go to https://itch.io/game/new
4. Set **Kind of project** to **HTML**
5. Upload the zip and check **"This file will be played in the browser"**
6. Set viewport dimensions to **800 x 540** (or adjust to taste)
7. Save and view your game page

## Project structure

```
TestWebRL/
├── index.html        # HTML entry point
├── package.json      # Dependencies and scripts
├── tsconfig.json     # TypeScript configuration
├── .gitignore
├── src/
│   └── main.ts       # Game code
└── dist/             # Built output (generated by npm run build)
```

## Notes

- Vite is pinned to v5 for Node 20.x compatibility. If you upgrade Node to 20.19+ or 22.12+, you can run `npm install vite@latest --save-dev` to move to Vite 7.
- rot-js docs: https://ondras.github.io/rot.js/manual/
